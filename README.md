## 설치 방법
### 게임 플레이

[게임 파일 배포 사이트](https://devel-rocket.itch.io/doongdoongisland "둥둥 아일랜드 다운받기")에 접속해 하단의 Download Now를 클릭해 게임 압축 파일을 다운받아주세요.  
압축 해제 후 폴더 내에 `DDIsand.exe` 파일을 실행해주세요.

### 프로젝트 파일

유니티 `6000.3.5f2` 버전으로 프로젝트를 실행해주세요.
